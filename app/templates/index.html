二， GIT提交Google&Qualcomm patch 的总结

1. cherry google/qcom提交

  • Cherrypick google的patch有一定的要求：
    1.必须保留原来的作者。
    2.在commit log里增加Mot-CRs-fixed: IKSWN-XXXX

    git cherry-pick
    push
    edit commit message : Mot-CRs-fixed: IKSWN-XXXX
  • 操作步骤如下：
    1 git cherry-pick --x commit id
    2 执行后
    2.1 没有冲突
    2.1.1 直接push上去：git push orgin HEAD:refs/for/xxxx
    2.2 有冲突
    2.2.1 解决冲突
    2.2.2 git cherry-pick --continue
    2.2.3 push上去：git push orgin HEAD:refs/for/xxxx
    2.2.4 提交信息中有冲突信息（手动解决冲突时残留的），记得删除（可在push到gerrit后再编辑删除）
